Entery Dashboard

Per questo progetto sono state utilizzate le seguenti tecnologie

Nome Tecnologia Tipologia Descrizione
Thymeleaf Motore di template Thymeleaf è un motore di template Java che permette di creare contenuti HTML dinamici per pagine web, consentendo di incorporare logica di visualizzazione direttamente nei template.
Data Jdbc Framework di accesso al database Data Jdbc è uno strumento di Spring che semplifica l’interazione con database relazionali, come MySQL, permettendo di eseguire operazioni SQL con metodi predefiniti e configurazioni minime.
Spring Web Modulo di sviluppo web Spring Web fornisce funzionalità per sviluppare applicazioni web e RESTful API, gestendo le richieste HTTP e la configurazione dell'infrastruttura per l’interazione con il frontend.
JavaMailSender Framework per l'invio di email JavaMailSender è parte di Spring e permette di inviare email tramite il protocollo SMTP, utilizzato in questo progetto per notifiche automatiche e conferme d’acquisto.
Materialize CSS Libreria di stile CSS Materialize CSS è una libreria frontend che implementa il Material Design di Google, utilizzata per migliorare l'aspetto grafico e la responsività dell'interfaccia utente.
Fetch API API di chiamate HTTP Fetch API è una funzione JavaScript per effettuare chiamate HTTP asincrone verso server esterni, utilizzata qui per ottenere dati da API e mostrarli nell'interfaccia.
